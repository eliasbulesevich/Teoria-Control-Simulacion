# PARA EJECUTAR SIMULADOR

## Opción 1: Descargar como ZIP (sin usar Git)

1. Hacé clic en el botón verde **Code**.  
2. Elegí **Download ZIP**.  
3. Extraé el archivo ZIP.  
4. Abrí el archivo `.html` en cualquier navegador (Chrome, Edge, Firefox, etc.).
